The Science of Forgetting

An interactive storytelling experience that explains how memory fades over time, using scroll-based narrative, animation, and data visualization to make a complex concept intuitive and engaging.

Overview

This project visualizes the Forgetting Curve, a concept from cognitive psychology that shows how information is lost over time without reinforcement.

Instead of presenting this as static content, the experience uses:
	•	Scroll-based storytelling
	•	Animated transitions
	•	Interactive data visualization

to guide the user through the concept step by step.

Key Features
	•	Narrative-driven design
Structured as a guided story, not a traditional app
	•	Interactive visualization
Users can adjust revision frequency and see how retention changes in real time
	•	Scroll-based animations
Sections animate into view to reinforce the flow of the story
	•	Dynamic feedback
Key metrics and explanations update based on user interaction
	•	Clean, editorial UI
Designed to feel like a modern interactive article

Tech Stack
	•	React (Vite)
	•	Framer Motion (animations)
	•	Recharts (data visualization)
	•	CSS / Tailwind (styling)
