# The Science of Forgetting

An interactive storytelling experience that explains how memory fades over time using scroll-based narrative, animation, and data visualization.

---

## 🧠 Overview

This project visualizes the **Forgetting Curve**, a concept from cognitive psychology that shows how information is lost over time without reinforcement.

Instead of presenting this as static content, the experience uses:

- Scroll-based storytelling  
- Animated transitions  
- Interactive data visualization  

to guide the user through the concept step by step.

---

## ✨ Key Features

- **Narrative-driven design**  
  Structured as a guided story, not a traditional app  

- **Interactive visualization**  
  Adjust revision frequency and see retention change in real time  

- **Scroll-based animations**  
  Sections animate into view to reinforce flow  

- **Dynamic feedback**  
  Metrics and explanations update based on interaction  

- **Clean, editorial UI**  
  Designed to feel like a modern interactive article  

---
